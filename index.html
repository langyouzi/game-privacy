【引言】

欢迎您体验《我独自飞升成仙》（下称“本游戏”）。我们深知个人信息对您的重要性，并将按照法律法规的规定，采取严格的安全保护措施，保护您的个人信息及隐私安全。本游戏为纯粹的单机游戏，我们致力于最大程度地减少信息收集，为您提供一个纯净的游戏环境。

本隐私政策将帮助您了解以下内容：

  1. 我们如何收集和使用个人信息
  2. 我们如何存储和保护个人信息
  3. 我们如何共享、转让、公开披露个人信息
  4. 您如何管理您的个人信息
  5. 未成年人保护条款
  6. 隐私政策的修订和通知
  7. 联系我们

一、我们如何收集和使用个人信息

为保障本游戏的正常运行和满足国家法律法规的合规要求，我们会在遵循合法、正当、必要原则的前提下，收集和使用您的个人信息。

1. 我们直接收集的信息

我们郑重承诺，本游戏本身不会以任何形式主动收集、存储或上传您的任何个人身份信息（例如：姓名、手机号、身份证号、电子邮件地址、地理位置、设备识别码IMEI/Android ID等）。

2. 为实现合规功能，由第三方收集的信息

为遵守国家新闻出版署关于网络游戏实名认证与防沉迷管理的相关规定，本游戏集成了 TapTap SDK。当您首次启动游戏时，会调用该SDK以完成实名认证和防沉迷流程。在此过程中：

  • 第三方名称：TapTap SDK
  • 用途和目的：用于游戏用户实名认证及国家防沉迷政策的落实。
  • 可能收集的个人信息类型：为完成实名认证，TapTap可能会收集您的身份信息（姓名、身份证号）及必要的设备信息。这些信息由您直接在TapTap的认证页面中提供，收集和处理遵循TapTap的隐私政策。
  • 数据流向：相关身份信息由TapTap官方服务收集和验证，仅用于其合规系统，我们自身不会以任何形式接触、存储或留存您的该部分信息。
  • 隐私政策链接：关于TapTap如何处理您的信息，请查阅其官方隐私政策。

3. 依法豁免征得同意收集和使用的个人信息

请您理解，在下列情形中，根据法律法规及相关国家标准，我们或我们的合作伙伴（TapTap）收集和使用您的个人信息无需征得您的授权同意：

  1. 与国家安全、国防安全直接相关的；
  2. 与公共安全、公共卫生、重大公共利益直接相关的；
  3. 与犯罪侦查、起诉、审判和判决执行等直接相关的；
  4. 出于维护个人信息主体或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；
  5. 所收集的您的个人信息是您自行向社会公众公开的；
  6. 用于维护本游戏的安全稳定运行所必需的，例如发现、处置产品的故障。

二、我们如何存储和保护个人信息

1. 信息存储地点

本游戏产生的所有数据（例如：游戏存档、设置等）均存储在您的本地设备上。我们不会将这些数据上传至任何服务器。

因实名认证与防沉迷需要，TapTap SDK收集的信息将存储于中华人民共和国境内，由TapTap负责存储。

2. 存储期限

由于游戏数据存储在您的本地设备上，其留存期限由您自己决定。当您卸载本游戏时，这些数据可能会被您的操作系统一并清除。对于TapTap收集的信息，其存储期限请参考TapTap的隐私政策。

3. 信息保护措施

我们非常重视您的信息安全。对于本地存储的游戏数据，其安全性依赖于您的设备安全环境。我们不会远程访问或操作这些数据。对于TapTap收集和处理的信息，TapTap会采用符合行业标准的安全措施进行保护。

三、我们如何共享、转让、公开披露个人信息

1. 共享、转让

我们郑重承诺，我们不会向任何第三方共享或转让您的个人信息。因为本游戏根本不收集您的个人信息。唯一的信息处理方是TapTap，其为实现合规目的而直接收集您的信息，此过程不涉及由我们向其共享或转让。

2. 公开披露

我们不会公开披露任何您的个人信息。

3. 收购、兼并、重组时的处理

若未来我们发生合并、收购、资产转តrANSFER等变更，我们将要求新的持有您个人信息的公司、组织继续受本隐私政策的约束，否则我们将要求该公司、组织重新向您征求授权同意。

四、您如何管理您的个人信息

我们非常重视您对个人信息的管理。

  • 对于游戏存档等本地数据：您可以在您设备的系统设置中，通过清除本游戏的应用数据或直接卸载本游戏的方式，来彻底删除所有本地数据。
  • 对于您提供给TapTap的实名认证信息：由于我们无法访问该信息，您需要根据TapTap的指引，在其平台或应用内进行查询、更正或提出其他请求。

五、未成年人保护条款

我们高度重视未成年人的信息保护。本游戏严格遵守国家防沉迷政策，通过接入TapTap实名认证与防沉迷系统来保护未成年人。

若用户被识别为未满18周岁的未成年人，游戏将严格遵循国家规定，仅在周五、周六、周日及法定节假日的20:00至21:00期间提供1小时服务。我们不会自行收集、存储或使用任何未成年人的个人信息。

我们建议未成年人在父母或其他监护人的指导下阅读本隐私政策。若您是未成年人的监护人，对您所监护的未成年人的个人信息有相关疑问时，请通过本政策公示的联系方式与我们联系。

六、隐私政策的修订和通知

为给您提供更好的服务，我们可能会根据服务的更新情况及法律法规的相关要求适时修订本政策的条款。当本政策发生变更时，我们会在版本更新时以弹窗或公告等形式向您展示变更后的政策。请您仔细阅读变更后的隐私政策，您继续使用本游戏即表示您同意受修订后的隐私政策约束。
